# Сеть R.E.A.C.T
Сеть REACT - это сеть сайтов, которые может добавлять каждый желающий пользователь Всемирной Паутины, независимо от региона и возраста. В сеть R.E.A.C.T входит:
1. Browser - упрощенная домашняя страница, с данными от всех сайтов.
2. Liternet - книжно-литературно сайт, в котором есть стихи как и классиков, как и пользователь Паутины.
3. Вконтакте Druza - сайт, основанный на старом дизайне сайта vkontakte.com, где пользователи и разработчики могут общаться и публиковать посты на стене.
10. Список будет обновляться ежемесячно. Все сайты доступны на kazah.ucoz.net/map.html. Там будут все сайты и их описания.
