# Сеть R.E.A.C.T
Сеть REACT - это сеть сайтов, которые может добавлять каждый желающий пользователь Всемирной Паутины, независимо от региона и возраста. В сеть R.E.A.C.T входит:
1. Browser - упрощенная домашняя страница, с данными от всех сайтов.
2. Liternet - книжно-литературно сайт, в котором есть сихи как и классиков, как и пользователь Паутины.
